对入境中国大陆（新疆除外）的外国人与港澳人所持Android手机上的安装的软件和保存的文件进行取证，有时也会用于对旅居海外的中国人进行取证。

所有取证工作均**针对Android端**，iOS端暂时只有机主解锁后人工浏览或使用简易数据恢复工具取证

*若机主不配合，地方政府会将情况上报，待上级批准后要求**Apple公司**与**云上贵州**提供**iCloud云端数据与App Store使用日志**
